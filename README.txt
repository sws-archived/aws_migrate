AWS Migrate
===========

NOTE: This module is deprecated, as most of its functionality has been replaced by the 7.x-2.x branch of the s3fs Drupal module, http://drupal.org/project/s3fs.

Drupal helper module to migrate to AWS S3 file hosting

Installing this module will convert your public file system handling to use the s3:// protocol

Uninstalling (not just disabling) this module will convert the s3:// protocol to public://
